简单介绍window操作

原始的Dstream：一个参数，表示多长时间划分一个RDD
窗口操作：两个参数，表示

  • window length（窗口长度）：窗口的持续时间（上图为3个时间单位）
  • sliding interval （滑动间隔）- 窗口操作的时间间隔（上图为2个时间单位）

对上图简单理解：每隔2个时间单位，对之前的3个时间单位操作
